# Students Performance Analysis
---

## 1 - Installation

```
pip install -r requirements.txt
```

## 2 - Setup Environment

- Create .env File
```
touch .env
```
- save openai key
```
OPENAI_API_KEY="PAST_YOUR_API_KEY_HERE"
```

## 3 - Run Project

- Go to the app directory
```
cd src/app/
```

- Run the Main file
```
python3 main.py
```
